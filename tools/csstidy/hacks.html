可用的 Hacks

下面是一个完整的 CSS hacks 列表，在默认设置下，CSSTidy 不会改变这些 hacks。如果启用“保护 CSS”，则会有更多的 hacks 被保留。一般来说，最好不要使用 CSS hacks，不过如果有需要，并且希望使用 CSSTidy，建议使用下面的 hacks 方法：

  • p\roperty:value; （仅在禁止删除不必要的反斜杠时有效）
  • div#test
  • head:first-child+body div
  • body>div
  • html[xmlns] div
  • @import "null?\"\{"; @import "styles.css"; （包括所有其他的 @import 规则）
  • @media all{/* rules */}
  • * html div
  • i{content:"\"/*"} div{property:value}
  • html*#test
  • _property:value
