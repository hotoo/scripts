Vim 自动补全成对的括号和引号

炫日分享了一个非常漂亮的脚本，来 自动补全成对的括号，【注】：原文中代码的引号被转义成了中文引号。 

inoremap ( ()<ESC>i
inoremap ) <c-r>=ClosePair(')')<CR>
inoremap { {}<ESC>i
inoremap } <c-r>=ClosePair('}')<CR>
inoremap [ []<ESC>i
inoremap ] <c-r>=ClosePair(']')<CR>
inoremap < <><ESC>i
inoremap > <c-r>=ClosePair('>')<CR>

function ClosePair(char)
    if getline('.')[col('.') - 1] == a:char
        return "\<Right>"
    else
        return a:char
    endif
endf

经此启发我增加了对引号的支持： 

inoremap ' <c-r>=CompleteQuote("'")<CR>
inoremap " <c-r>=CompleteQuote('"')<CR>

function CompleteQuote(quote)
    if getline('.')[col('.') - 1] == a:quote
        return "\<Right>"
    else
        return a:quote . a:quote . "\<Left>"
endfunction

不过中文全角的括号和引号目前无法通过映射来实现，如果有好的实现也请告知 : )
