用 Vimwiki + Dropbox 写博客

由于众所周知的缘故，我的博客 被消失了。在国内买域名或主机都是十分不靠谱的事，所以目前没有重新开独立博客的打算。而使用第三方博客服务也同样不靠谱，这也不在计划之内。

把东西都放在自己的计算机里算是比较可靠的方式了，我计划使用 Vim + Vimwiki + Dropbox 来搭建自己的特殊博客系统。

使用 Vim 这个极致的编辑器，加上 Vimwiki 这个已经很好，并且还在更好的插件，使用 wiki 语法来写日志，然后由 Dropbox 来同步与发布。整体上来说这个系统基本完好，除了域名稍稍缺憾之外。

如果你看过 Vim Script 文档中文计划 的介绍，下面有写内容大概已经熟悉，这里作为完整的技巧一并介绍：

  • 确认正确安装了 Vim 和 Vimwiki。
  • 如果安装了 snipMate(可选) 在 $VIM/vimfiles/snippets/ 下新建一个 vimwiki.snippets 文件（已有这打开文件）。在 vimwiki.snippets 中添加自己的博客格式模板，例如我的： 
snippet blog
    = ${1:title} =
    ${2:content}

     blog:datetime=${3:`strftime("%Y/%m/%d %H:%M:%S")`}:tags=${4}
  • 因为 Vimwiki 和 snipMate 的 <Tab> 热键冲突，可以将 $VIM/vimfiles/ftplugin/vimwiki.vim 中的第 293 行的 
inoremap <expr> <buffer> <Tab> vimwiki_tbl#kbd_tab()

换成其他的热键，例如： 

inoremap <expr> <buffer> <C-Tab> vimwiki_tbl#kbd_tab()
  • 到此，在 .wiki 文件中输入 blog<Tab> 就展开博客模板了。
  • 在 vimrc 的 g:vimwiki_list 中加入一个 wiki，例如我的： 
let g:vimwiki_list = [
                \ {...},
                \ {...},
                \ {...},
                \ {
                    \ 'path'        : 'D:\My Dropbox\blog',
                    \ 'path_html'   : 'D:\My Dropbox\Public\blog',
                    \ 'html_header' : 'D:\My Dropbox\blog\template\header.tpl',
                    \ 'html_footer' : 'D:\My Dropbox\blog\template\footer.tpl'
                \ }
                \ ]

这样我使用 4<leader>ww(一般是4\ww) 就可以快速启动博客 wiki 首页并进行编辑了。wiki 文件保存在 Dropbox 私有目录下，生成的 html 文件保存在公开目录，这样，草稿日志不转成 html 就可以了，写好想发布、或想重新发布，:Vimwiki2HTML 即可。
