move

Object move(<Number,> t, <Number> i)
    标签集内页签位置调整，将第一个参数所引用的页签移至第二个参数指定的索引位置处。
    Parameters:
      t - 指定被移动页签对象。 如果是数值型(Number)，则引用页签集对应索引位置的页签对象； 如果是页签类型(TabPage)，且页签存在于页签集合中，则直接引用之； 否则，抛出异常。
      i - 指定目标索引。
