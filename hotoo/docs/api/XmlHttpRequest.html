Constructor Detail

XmlHttpRequest

XmlHttpRequest(<Boolean> a)
    封装的兼容多浏览器的XMLHttpRequest（注意大小写）对象。隐藏了各浏览器间异步&同步方式的差异性。
    Parameters:
      a - 是否使用异步方式。false则为同步，否则异步。
    Version: 2006/4/16, 2008/2/29

    Author: 闲耘 (mail@xianyun.org)
    See:
      - 参考Robin Pan (htmlor@gmail.com)的同名类。

send

void send(u, m, p, c)
    发送请求。
    Parameters:
      url - 目标请求地址，可以是绝对/相对路径。
      method - 发送请求的方式，"post"/"get"。默认为"get"方式。
      param - 发送请求中带的参数。
      callback - 发送请求过程中的回调函数。
